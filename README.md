# Simple CRM

This is a very simple CRM for beginners.

## What it does

- Add clients
- Edit clients
- Delete clients
- Search clients
- Save data in the browser using `localStorage`

## Client fields

- City
- Store name
- Phone
- Notes
- Next visit date
- Client type

## How to use it

1. Open `index.html` in your browser.
2. Fill in the form.
3. Click `Save client`.
4. Your clients will appear in the table below.

## Important

The data is saved only in your browser on your computer. If you clear browser data, the saved clients may be removed.
